## IN
- Returns True if an element exists inside an object

Example:
```python
users = ['James', 'Charlie', 'Ana']
if 'James' in users:
    print(True)
```

## NOT IN
- Returns True if an element does not exist inside an object

Example:
```python
users = ['James', 'Charlie', 'Ana']
if 'Johnny' in users:
    print(True)
```
